🃏 Blackjack Game (Python)


📌 About This Project...
This is a simple Blackjack game I built using Python while learning how loops, conditions, and functions work together.
It’s basically a console version of Blackjack where you play against the computer. You try to get as close to 21 as possible without going over.

🎮 How it works...
You and the computer are dealt 2 cards each
You can choose to:
Draw another card (y)
Stop and pass (n)
If your total goes over 21 → you lose (busted 💀)
Computer also plays and compares scores
Winner is decided based on who is closer to 21

🛠️ Tech used...
Python
Random module

🧠 What I learned from this project...
How loops (while True) actually control game flow
Using functions to handle logic (like calculating total score)
Handling conditions and game rules
Working with lists and modifying values (like Ace = 11 → 1)
Building simple game logic step by step

🎯 Game Rules I implemented...
Cards: 2–10 + face cards (treated as 10) + Ace (11 or 1)
Ace automatically changes from 11 → 1 if needed
Score comparison after each round
Bust logic if score > 21
Option to replay the game
